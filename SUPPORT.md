# Support

## Community Project

This is a **community-maintained** open-source project. Support is provided on a **best-effort basis** — there is no SLA or guaranteed response time.

## Getting Help

| Channel                                 | Use For                                 |
| --------------------------------------- | --------------------------------------- |
| [GitHub Issues](../../issues)           | Bug reports and feature requests        |
| [GitHub Discussions](../../discussions) | Questions, ideas, and community support |
| [README](README.md)                     | Module documentation, usage examples    |
| [Examples](examples/)                   | Working example configurations          |
| [Tests](tests/)                         | Test patterns and usage references      |

## Before Opening an Issue

Please check the following before opening a new issue:

1. **Search existing issues** — Your question may already be answered.
2. **Check the README** — Review the module documentation for usage guidance.
3. **Review the examples** — The `examples/` directory contains working configurations.
4. **Verify your versions** — Ensure you're using supported versions of Terraform and the Power Platform provider.
5. **Minimal reproduction** — If reporting a bug, include a minimal Terraform configuration that reproduces the issue.
